## How to Use

### Opening the AI Assistant

1. Look for the **floating robot icon (🤖)** in the bottom-right corner
2. Click it to open the AI Assistant panel
3. Start chatting with the bot!

### Sending Messages

**Type a message** in the input field and:
- Press **Enter** to send
- Click the **paper plane icon** to send
- Hold Shift+Enter for multi-line messages

### Quick Actions

The top of the chat panel has 4 quick buttons:

| Button | What it Does |
|--------|-------------|
| 🆘 Get Help | General assistance |
| 💡 Chat Tips | Conversation advice |
| ⭐ Features | App feature tour |
| 📧 Contact Dev | Developer info |

Click any button for instant answers!

## Tips for Best Results

### How to Get Better Answers

1. **Be specific** - "How do I enable Couple Mode?" works better than "Couple Mode?"
2. **Ask one thing at a time** - Focus on one topic per message
3. **Use keywords** - Include words like "feature," "help," "tips," "contact"
4. **Provide context** - Mention what you're trying to do
5. **Be clear** - Avoid slang or abbreviations

### Example Good Questions

✅ "How do I change my profile avatar?"  
✅ "What's the difference between themes?"  
✅ "How do Love Meters work in Couple Mode?"  
✅ "I'm having trouble sending messages, what should I do?"  
✅ "How do I contact the developer with a bug report?"  

### Example Questions to Avoid

❌ "lol help"  
❌ "???"  
❌ "idk what im doing"  
❌ "everything is broken"  
❌ Just random text

## Common Questions Answered

### Q: How do I use Couple Mode?
**A:** Click the ❤️ button in the chat header. The AI can walk you through the setup!

### Q: Can I schedule messages?
**A:** Yes! Use the calendar icon (📅) in the chat features. Ask the AI for details!

### Q: How do secret messages work?
**A:** They appear encrypted in chat. Use them for private moments. Ask the AI for the full guide!

### Q: What's the Love Meter?
**A:** It tracks connection in Couple Mode - increases with every message! Ask the AI to explain.

### Q: Can I use different themes?
**A:** Yes! Click the palette icon (🎨) to choose themes. The AI can explain all of them!

### Q: How do I report a bug?
**A:** Ask the AI for developer contact info, then reach out via email!

### Q: Is my data private?
**A:** Yes! ChatSphere uses end-to-end encryption. Check the login screen info.

### Q: How do I delete messages?
**A:** Currently, messages can't be deleted, but you can use secret messages for privacy!

## Troubleshooting the AI Assistant

### AI not responding?

1. **Check connection** - Ensure backend server is running
2. **Refresh page** - Reload the app (Ctrl+R)
3. **Try again** - Send message again after refresh
4. **Check console** - Open DevTools (F12) for errors

### Button not working?

1. Refresh the page
2. Clear browser cache
3. Try different browser
4. Contact developer

### Messages not showing?

1. Check panel is scrolled to bottom
2. Send a new message
3. Refresh the app
4. Restart backend server

### For Developers

const openai = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

const completion = await openai.chat.completions.create({
  model: "gpt-4",
  messages: [
    { role: "system", content: "You are ChatSphere AI assistant..." },
    { role: "user", content: userMessage }
  ]
});

return completion.choices[0].message.content;
